# Aura - Cyber-Premium Resume Analyzer

![Aura Scan UI](https://raw.githubusercontent.com/lucide-icons/lucide/main/icons/zap.svg)

**Aura** is a next-generation, professional-grade resume auditing platform. Built with a signature **"Cyber-Premium"** aesthetic, it transforms static PDF resumes into high-fidelity diagnostic dashboards. 

By leveraging the **Gemini 2.5 Pro** engine alongside a custom **Intelligent Heuristic Protocol**, Aura provides deeper insights than standard scanners, ensuring your career brand is optimized for both human elite recruiters and machine ATS algorithms.

---

## ✨ Key Features

### 1. Unified Diagnostic Engine
Aura doesn't just "score" your resume; it performs a clinical-grade audit of your career trajectory.
- **Neural Analysis**: Powered by Gemini for deep contextual insights.
- **3-Phase Action Roadmap**: A strategic, interactive blueprint for your next 30, 60, and 90 days.
- **Executive Verdict**: Real-world compatibility checks identifying semantic failure points.

### 2. Intelligent Protocol (Zero-Downtime Resilience)
Aura is designed for 100% mission availability.
- **Auto-Fallback**: If the AI engine is under high demand (503), Aura activates its **Integrated Heuristic Engine** to provide a data-driven report based on your specific tech stack and metrics.
- **Retry Logic**: Built-in exponential backoff ensures that temporary network spikes never stop your audit.

### 3. Cyber-Premium Dashboard
A workspace designed for high-density information architecture.
- **Bento Grid Analytics**: Targeted insights for Skills, Impact, Structure, and Visual Logic.
- **Live Persistence**: Your analysis results and Job Descriptions are saved to `localStorage`, so you never lose progress on a page reload.
- **Context-Aware Tools**: Generate **Tailored Cover Letters** and **ATS Keyword Matrixes** directly from your dashboard.

---

## 🛠 How It Works

1. **Fiber-Optic Parsing**: Aura uses a high-performance PDF worker to deconstruct your document into a pure data matrix.
2. **Context Injection**: By providing an (optional) Job Description, the engine calibrates its audit to precisely measure your "delta" against specific role requirements.
3. **Neural Auditing**: The text is processed through Gemini with a specialized "Recruiter Rubric" prompt that focuses on quantification (%, $), active narrative, and tech-stack density.
4. **Heuristic Sanitization**: Results are normalized through our fallback layer to ensure you always get actionable advice, even during API downtime.

## ⚙️ Setup & Installation

### Prerequisites
- Node.js 18+
- [Google Gemini API Key](https://aistudio.google.com/app/apikey)

### Quick Start
1. **Clone & Install**:
   ```bash
   git clone https://github.com/PriyanshiGoyal-15/Aura-Resume_Analyzer.git
   cd Aura-Resume_Analyzer/my-app
   npm install
   ```

2. **Environment Configuration**:
   Create a `.env.local` file in the root:
   ```env
   GEMINI_API_KEY=your_key_here
   ```

3. **Launch**:
   ```bash
   npm run dev
   ```
